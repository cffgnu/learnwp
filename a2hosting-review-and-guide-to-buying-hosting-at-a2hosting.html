Turbo server

Turbo server for better page load speed thanks to:

  • There are few users sharing resources on the physical server
  • More memory than other plans (2G RAM vs 1GB specifically)
  • A2 Optimized Site Accelerator – supports server-level preconfigured cache (using Turbo cache, OPcache, and Memcached)

A2 Optimized Site Accelerator

A2 Optimized Site Accelerator is a great feature included in the Turbo Plan. It is a cPanel plugin that provides preconfigured caching for your website. You can find the following cache options:

  • Turbo Cache: all HTML content is stored in Turbo Cache and returned to the client without having to run PHP. As you know PHP code processing is always the slowest when it comes to website loading time.
  • OPcache/APC: Testing shows that OPcache cuts PHP page response time in half.
  • Memcached: speed up database access by keeping important data in memory for faster queries.

HTTP/2

Turbo server supports HTTP/2. HTTP/2 is an important update to HTTP/1.1

In fact, HTTP/2 offers faster page load times than HTTP/1.1 because it is capable of handling multiple HTTP requests at a time.

Server Rewind

A2 Hosting’s cPanel provides you with the Server Rewind tool. Server Rewind protects your website from loss of important data due to accidental deletion, for example. You can easily restore your website when needed. 

Good security

Another highlight of A2Hosting is its very good security. Your website is very difficult to be hacked when using A2 Hosting’s hosting service. This is due to A2 Hosting’s 24/7/365 security monitoring system.

In addition, A2 Hosting is also integrated into the Patchman cPanel service. Patchman automatically detects malware and security holes in your website.

As soon as they are detected, Patchman will email you a notification. You can handle it manually or let patchman automatically patch the security of your website.

WordPress version with A2 Optimized pre-installed

You have many options to install WordPress on A2Hosting

  • Install WordPress right from the hosting registration step. Installing WordPress couldn’t be simpler. As soon as you purchase a hosting package, A2Hosting will send you an email containing your login information.

Change PHP version

By default, your website runs with PHP 5.5. You should switch to PHP 7 so you get the best website performance.

Go to cPanel. Scroll down to the Software section. Click on the link Select PHP version.

On the next screen, select version 7.0 from the drop-down menu. And click the Set as cure button.

Next, you choose opcache and memcached. Remember to click the Save button to save your settings.

Enable Memcached and TurboCache

By default, Memcached and TurboCache are not enabled. To enable it, go to cPanel. Scroll down to the Advanced section, and click on A2 Optimized.

Next, click on Memcached in the left menu. Then click Enable.

Finally, click on Turbo Cache and click on Enable similar to Memcached. ‘

Using Widgets

Using Woocommerce widgets is the same as using other widgets. Which Widget you want to use just drag to the sidebar you want to display. As usual, you will configure at Appearance -> Widgets

List of Woocommerce Widgets

1. WooCommerce Cart (WooCommerce Cart)

 Display mini cart, help customers see the list of products currently in the cart

2. WooCommerce Layered Product Guide (WooCommerce Layered Nav)

Allow customers to filter products by attribute. The widget is only visible in the product archive page (store page).

Options of this Widget

Title : The title displayed on the outside of the Widget

Attribute : Select the product’s attribute to filter. The value here is the product slug so it’s unsigned.

Display Style : Choose how the instructions are displayed.

Query type :

  • And: show only products whose attributes match your preferences. Suppose if you select 2 attributes, only products with both attributes will appear
  • OR: If you select 2 attributes, products that match either attribute will appear.

3. WooCommerce Layered Nav Filters

Display the properties you are filtering the product view with the Woocommerce Layered Nav widget. The purpose is to let customers know which property they are filtering by to remove it

4. Woocommerce Price Filter (Woocommerce Price Filter)

The filter allows to filter products by price by using a slide to select the price range. This widget is only visible in the Store page (product archive page)

5. WooCommerce Product Categories

Display the list of existing products on the website

6. WooCommerce Products (WooCommerce Products)

Options of this Widget

Title : The Widget’s display title outside the interface.

Number of products displayed : Customize the maximum number of products you want to display to the outside.

Display

  • All products: Displays all products in a customized sort order below.
  • Featured Products : Display featured products in a customized sort order below.
  • Discounted Products : Displays discounted products in a customized sort order below.

Sorted by

  • Date : Sort products by date
  • Price : Sort products by price
  • Random : Sort products in a random fashion
  • Sales : Sort products by sales.

Order

  • Descending : Sort products in descending order
  • Ascending : Sort products in ascending order.

7. Product Search in Woocommerce (WooCommerce Product Search)

Show search box and search for products only

8. Product Keywords in Woocommerce (WooCommerce Product Tags)

Display product keywords in cloud format

9. Recent Reviews in Woocommerce (WooCommerce Recent Reviews)

Show reviews in the latest products, with the number of review stars

10. Recently viewed in WooCommerce (WooCommerce Recent Review)

Display the products the customer has just viewed

11. The Highest Rated Products in WooCommerce (WooCommerce Top Rated Products)

List of products with the most user reviews.

Partitioning the Sidebar display with WooSidebars

A fairly common problem when you use Widgets in Woocommerce. That is, you need to divide the displayed Widgets into different pages.

Let’s say you only have a single sidebar showing on the right. But you want the product display page to have its own widget, the product category page to have its own widget.

To solve this problem, you can use the free plugin WooSidebars . This plugin supports you to create your own sidebar and replace the existing sidebar.

Now let’s see how to use WooSidebars.

Using WooSidebars

You install the plugin just like any other client plugin in WordPress. Once installed, you will notice a new item called Widget Area in the Appearance menu. This section helps you create sidebars to replace the existing sidebar

First, click Add New to create a new sidebar.

You enter a name for the sidebar. In the Conditions section, you choose the sidebar display conditions. You can choose Advanced to display the full interface with many advanced options.

Most importantly, in the Side To Replace section, you choose the default side to replace when the condition in Conditions is satisfied. After you finish configuring, click Post.

Next, go to Appearance -> Widgets , you will see the sidebar you just created. The task now is to drag the widget to display to the sidebar. So the widgets will display according to the logic we just created above. Simple right?
